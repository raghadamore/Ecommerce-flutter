# 🛒 E-Commerce Flutter App

A full-featured e-commerce mobile application built with Flutter, Firebase, and GetX following the MVVM architecture. The app provides a complete shopping experience including authentication, product browsing, cart management, wishlist, and user profile management.

---

## 🚀 Features

### 🔐 Authentication
- Email & Password Login  
- Google Sign-In  
- Sign Up with Email  
- Forgot Password  
- Auto Login using SharedPreferences  

---

### 🏠 Home Screen
- Display featured sections  
- Show a summary of products  
- Navigate to product details  
- Clean and responsive UI  

---

### 🏬 Store Screen
- Display all brands  
- Each brand includes:
  - Brand summary  
  - Number of products  
- View products by brand  
- Filter products by categories/types  
- Browse all brands and products  

---

### 📄 Product Details
- View detailed product information  
- Select:
  - Quantity  
  - Color  
  - Variants/options  
- Add to Cart 🛒  
- Add to Wishlist ❤️  

---

### 🛒 Cart
- Add/remove products  
- Increase/decrease quantity  
- View selected items  
- Manage delivery addresses  
- Update cart items dynamically  

---

### ❤️ Wishlist
- Add/remove favorite products  
- View saved items  
- Add items to cart  

---

### 👤 Profile
- View and edit personal information  
- App settings  
- Manage addresses:
  - Add new address  
  - Edit existing addresses  
- Access My Cart and My Addresses  

---

## 🔥 Persistence
- User session saved using SharedPreferences  
- Auto-login implemented after authentication  

---

## 🧰 Tech Stack

- Flutter  
- Firebase Authentication  
- Cloud Firestore  
- Firebase Storage (optional)  
- GetX (State Management)  
- SharedPreferences
